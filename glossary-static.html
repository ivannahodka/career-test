Глоссарий профессиональных терминов

753 термина из IT, медицины, науки, инженерии, психологии, фармации и педагогики — на русском и английском языке.

IT

API
Application Programming Interface — интерфейс для общения между программами через интернет
SPA
Single Page Application — сайт который не перезагружается целиком, как Gmail или Trello
CSS
Cascading Style Sheets — описывают как выглядит страница: цвета, шрифты, отступы
HTML
HyperText Markup Language — язык разметки: описывает структуру страницы (заголовки, кнопки, ссылки)
SQL
Structured Query Language — язык запросов к базам данных. Пример: SELECT * FROM users
REST
Representational State Transfer — архитектурный стиль для API: как именно программы общаются
CI/CD
Continuous Integration / Delivery — автоматический тест и деплой кода при каждом изменении
SRE
Site Reliability Engineering — инженеры которые следят чтобы сервис работал 24/7 без падений
IaC
Infrastructure as Code — серверы и сети описываются кодом, а не настраиваются руками
SLO
Service Level Objective — цель по надёжности: например сайт доступен 99.9% времени
ORM
Object-Relational Mapping — библиотека которая позволяет работать с БД без написания SQL
Agile
Гибкая методология разработки: работа короткими итерациями (спринтами) с частыми обновлениями
Scrum
Фреймворк Agile: спринты по 2 недели, ежедневные стендапы, ретроспективы в конце спринта
TypeScript
Надстройка над JavaScript которая добавляет строгие типы — код становится надёжнее
Redux
Библиотека управления состоянием для React — хранит все данные приложения в одном месте
Backlog
Список задач продукта упорядоченный по приоритету
DevOps
Культура и практики объединяющие разработку и операции — цель: быстрый и надёжный деплой
Docker
Инструмент для запуска приложений в изолированных контейнерах — работает одинаково везде
Kubernetes
Система управления контейнерами Docker в большом масштабе — оркестрация
Pixel-perfect
Реализация дизайна точно как у дизайнера: каждый пиксель на своём месте
Git
Система контроля версий — записывает историю изменений кода, позволяет работать в команде
Linux
Операционная система с открытым кодом — основа большинства серверов и DevOps-инструментов
Python
Язык программирования №1 для Data Science, ML и автоматизации — очень простой синтаксис
JavaScript
Язык программирования браузера — делает сайты интерактивными; также работает на сервере (Node.js)
React
JavaScript-библиотека от Meta для создания UI — самый популярный фронтенд-фреймворк
Vue
JavaScript-фреймворк для создания UI — популярная альтернатива React, особенно в Азии и Европе
Zustand
Маленькая и простая библиотека состояния для React — альтернатива Redux без лишней сложности
GitHub
Платформа для хранения кода и совместной разработки на основе Git
GraphQL
Альтернатива REST для API — клиент сам указывает какие данные ему нужны
WebSocket
Протокол для двустороннего соединения в реальном времени — чаты, игры, трейдинг
PWA
Progressive Web App — веб-приложение которое работает как нативное: офлайн, иконка на экране
SSR
Server-Side Rendering — страница генерируется на сервере, быстрее загружается и лучше для SEO
Kanban
Метод визуализации задач: колонки «Сделать», «В процессе», «Готово» — как Trello
Wireframe
Скелет интерфейса — схематичный макет без цветов и шрифтов, только структура
Codeforces
Платформа соревновательного программирования — задачи на алгоритмы и структуры данных
LeetCode
Платформа для подготовки к техническим собеседованиям — задачи по алгоритмам
Jira
Система управления задачами — команды ведут в ней баги, задачи и спринты
GitHub Actions
Встроенный CI/CD в GitHub — автоматически запускает тесты и деплой при пуше кода
Grafana
Инструмент для мониторинга и дашбордов — показывает графики нагрузки серверов в реальном времени
Telegram
Мессенджер с мощным Bot API — канальная экосистема, популярная в русскоязычном интернете
Discord
Мессенджер популярный у геймеров и разработчиков — есть голосовые каналы и боты
Стендап
Ежедневная 15-минутная встреча команды: что сделал, что сделаю, что мешает
Спринт
Итерация в Scrum — обычно 1–2 недели, в конце должна быть рабочая версия

Дизайн

UI

UX

Показываешь двум группам пользователей разные версии и смотришь какая работает лучше

Творческие

CMYK
Cyan Magenta Yellow Key — цветовая модель для печати. Отличается от RGB: печать и экран передают цвет по-разному
RGB
Red Green Blue — цветовая модель для экранов. Все цвета получаются смешением трёх каналов света
RAW
Необработанный файл с камеры — содержит максимум информации, требует обработки в Lightroom или Capture One
AIDA
Attention Interest Desire Action — классическая формула копирайтинга: привлечь внимание, вызвать интерес, желание, действие
CTA
Call To Action — призыв к действию: кнопка, заголовок или фраза которая говорит пользователю что делать дальше
ISO
Чувствительность матрицы камеры к свету. Высокое ISO = больше шума, низкое = чистое изображение
ArtStation
Профессиональная платформа-портфолио для художников, иллюстраторов и концепт-артистов
Storyboard
Раскадровка — серия рисунков-эскизов, показывающих как будет выглядеть сцена видео или мультфильма
Reels

Вектор
Изображение описанное математикой (кривые Безье) — масштабируется без потери качества. Формат: SVG, AI, EPS
Растр
Изображение из пикселей — при увеличении теряет качество. Форматы: JPG, PNG, PSD
Мудборд
Mood board — коллаж из референсов: фото, цвета, текстуры, шрифты. Показывает направление стиля до начала работы
Гайдлайн
Brand Guidelines — документ с правилами использования фирменного стиля: логотип, цвета, шрифты, отступы
Кернинг
Расстояние между отдельными буквами в тексте — важно для читаемости и эстетики типографики
Макет развёртки
Dieline — плоский шаблон упаковки с линиями реза и сгиба. По нему печатают и собирают коробку
SEO-текст

Охват
Reach — количество уникальных людей которые увидели публикацию. Отличается от просмотров
Вовлечённость
Engagement Rate — процент аудитории которая взаимодействует с контентом: лайки, комменты, репосты
Контент-план
Редакционный план публикаций: темы, форматы, даты — помогает вести соцсети системно
Хук
Hook — первые 3 секунды видео или первая фраза текста которая цепляет и не даёт уйти
Turnaround
Разворотная схема персонажа — вид спереди, сбоку и сзади для аниматоров и 3D-моделлеров
Боке
Bokeh — размытый фон на фотографии. Достигается широкой диафрагмой (малое f-число)
Экспозиция
Количество света попавшего на матрицу — зависит от выдержки, диафрагмы и ISO
Ретушь
Обработка фотографии: удаление дефектов, выравнивание кожи, коррекция цвета в Photoshop
Цветокоррекция
Color Grading — художественная обработка цвета: создание настроения через тёплые/холодные тона

Raster
Растр — изображение из пикселей, теряет качество при увеличении. Форматы: JPG, PNG, PSD
Moodboard
Мудборд — коллаж из референсов: фото, цвета, текстуры, шрифты. Показывает стиль до начала работы
Brand Guidelines
Гайдлайн бренда — документ с правилами использования визуальной айдентики
Kerning
Кернинг — настройка расстояния между отдельными буквами в типографике

SEO copy
SEO-текст — контент, оптимизированный под поисковые запросы для попадания в топ Google
Reach
Охват — число уникальных людей, увидевших публикацию
Engagement Rate

Медицина и наука

ЭМК
Электронная медицинская карта — цифровая история болезни пациента в информационной системе клиники
МКБ-10
Международная классификация болезней 10-го пересмотра — единый язык диагнозов для врачей всего мира
Анамнез
История болезни: жалобы, симптомы, перенесённые заболевания — собирается врачом при первичном приёме
Дифференциальный диагноз
Методический процесс исключения болезней со схожими симптомами чтобы прийти к единственному верному диагнозу
КПТ
Когнитивно-поведенческая терапия — научно доказанный метод психотерапии для лечения тревоги и депрессии
РКИ
Рандомизированное контролируемое исследование — золотой стандарт клинических испытаний: пациентов случайно делят на группы
ПЦР
Полимеразная цепная реакция — метод анализа ДНК и РНК; используется для диагностики инфекций и генетических заболеваний
CRISPR
Технология редактирования генома — позволяет точечно изменять участки ДНК; применяется в биотехнологии и медицине
Секвенирование
Определение последовательности нуклеотидов в ДНК или РНК — основа персонализированной медицины и биоинформатики
Биоинформатика
Наука на стыке биологии и программирования: анализ геномных данных, поиск по базам BLAST/NCBI, структуры белков
Биомаркер
Измеримый биологический показатель (уровень белка, генетическая мутация), указывающий на болезнь или эффект лечения
Плацебо
Препарат-пустышка без активного вещества — используется в клинических исследованиях как контрольный образец
GCP
Good Clinical Practice — международный стандарт проведения клинических исследований, защищающий права участников
Фармакокинетика
Как лекарство движется по организму: всасывание, распределение, метаболизм, выведение (ADME)
PubMed
Крупнейшая бесплатная база медицинских и биологических научных статей от Национальной библиотеки медицины США
BLAST
Basic Local Alignment Search Tool — программа поиска похожих последовательностей ДНК и белков в базах NCBI
Доказательная медицина
Принятие клинических решений на основе лучших доступных научных данных, а не только опыта врача
UpToDate
Медицинская база знаний с клиническими рекомендациями — врачи используют её для принятия решений у постели больного
Benchling
Облачная платформа для ведения лабораторных журналов и управления биотех-экспериментами
BioPython
Набор Python-инструментов для биоинформатики: работа с последовательностями ДНК, парсинг BLAST, анализ структур
ATLS
Advanced Trauma Life Support — международный курс по оказанию первичной помощи при травмах; обязателен для хирургов и врачей скорой помощи
ACLS
Advanced Cardiac Life Support — курс по расширенной реанимации при остановке сердца и неотложных кардиологических состояниях
FHIR
Fast Healthcare Interoperability Resources — современный стандарт обмена медицинскими данными через REST API; замена устаревшим форматам
HL7
Health Level 7 — семейство стандартов для обмена и хранения медицинской информации между системами; HL7 v2 до сих пор используется в большинстве клиник
Лапароскоп
Хирургический инструмент: видеокамера и манипуляторы вводятся через маленькие проколы — пациент быстрее восстанавливается, чем после открытой операции
Эндоскоп
Гибкая трубка с камерой для осмотра внутренних органов (желудок, кишечник, бронхи) без разреза
Wet lab
Учебная лаборатория для отработки хирургических навыков на биологических тканях или трупном материале — основа подготовки хирургов
Индекс Хирша
h-index — наукометрический показатель: учёный с h=20 опубликовал ≥20 статей, каждая из которых цитировалась ≥20 раз
Телемедицина
Оказание медицинской помощи дистанционно — видеоконсультации, удалённый мониторинг пациентов через носимые устройства
Роботизированная хирургия
Операции с помощью робота da Vinci — хирург управляет манипуляторами с высокой точностью; меньше осложнений, быстрее восстановление
Мотивационное интервьюирование
Метод консультирования, помогающий пациенту самостоятельно прийти к решению изменить поведение — отказаться от курения, похудеть и т.д.
Мультидисциплинарная команда
MDT — группа специалистов разных профилей (врач, хирург, онколог, психолог, медсестра), совместно ведущих сложного пациента
Ординатура
Клиническая резидентура — 2–6 лет специализированной практики под руководством врачей после получения диплома
MedTech
Medical Technology — отрасль технологий для медицины: диагностические устройства, носимые датчики, ИИ для анализа снимков, электронные рецепты
Журнал Q1
Научный журнал первого квартиля по импакт-фактору — публикация там означает высокое качество рецензирования и широкую читаемость
Cochrane
Cochrane Library — крупнейшая база систематических обзоров и мета-анализов: синтезирует лучшие доказательства для клинической практики
Биостатистика

Стандартный текстовый формат хранения последовательностей NGS-чтений вместе с оценками качества каждого нуклеотида; исходный файл любого геномного анализа

Variant Call Format — стандартный формат хранения генетических вариантов (SNP, инделов) относительно референсного генома; используется в клинической геномике
Fermentation

Психология

ПТСР
Посттравматическое стрессовое расстройство — психическое расстройство после тяжёлого травматического события; симптомы: флэшбэки, кошмары, избегание, гипербдительность
MMPI
Minnesota Multiphasic Personality Inventory — самый распространённый стандартизированный личностный тест; выявляет психопатологию и черты личности по 10 клиническим шкалам
Роршах
Проективный тест Роршаха — 10 симметричных чернильных пятен, по интерпретации которых клинический психолог оценивает мышление, восприятие и эмоциональное состояние
МКБ-11
Международная классификация болезней 11-го пересмотра (ВОЗ) — глобальный стандарт диагностики всех заболеваний, включая психические расстройства
DSM-5
Диагностическое и статистическое руководство по психическим расстройствам 5-го издания (APA) — стандарт диагностики в США и основа большинства исследований
EMDR
Eye Movement Desensitization and Reprocessing — метод психотерапии травмы: клиент вспоминает травмирующее событие, следя за движениями руки терапевта; доказан для ПТСР
Схема-терапия
Интегративный метод психотерапии Джеффри Янга — работает с глубинными убеждениями (схемами) сформированными в детстве и лежащими в основе хронических проблем
Патопсихология
Раздел клинической психологии, изучающий нарушения психических процессов (памяти, мышления, восприятия) при психических расстройствах; использует экспериментальные методики
Нейропсихология
Наука о связи мозга и поведения: оценивает когнитивные функции (память, внимание, речь, праксис) при повреждениях мозга — инсульте, ЧМТ, деменции
фМРТ
Функциональная МРТ — метод нейровизуализации, показывающий активность областей мозга в реальном времени по изменению кровотока; стандарт в когнитивной нейронауке
SPSS
Statistical Package for the Social Sciences — программа для статистического анализа данных; стандарт в психологических исследованиях: корреляции, дисперсионный анализ, факторный анализ
HOGAN
Серия личностных опросников Hogan Assessments — оценивает потенциал, риски и ценности кандидатов; используется в подборе и развитии лидеров в крупных компаниях
SHL
Британская компания и система тестов для оценки персонала: тесты способностей (вербальных, числовых, диаграммных) и личностный опросник OPQ
eNPS
Employee Net Promoter Score — показатель лояльности сотрудников: один вопрос «Порекомендуете ли вы компанию как работодателя?»; от -100 до +100
Gallup Q12
12 вопросов Gallup для измерения вовлечённости сотрудников — один из самых валидных инструментов оценки рабочей среды; результаты коррелируют с производительностью
ICF
International Coaching Federation — крупнейшая международная организация профессиональных коучей; выдаёт аккредитации ACC, PCC, MCC; стандарт качества в коучинге
Супервизия
Регулярные встречи психолога с более опытным коллегой-супервизором для разбора случаев из практики и профессионального развития; обязательна в психотерапии
Ассессмент
Assessment Center — метод оценки кандидатов или сотрудников через набор упражнений (кейсы, ролевые игры, интервью), наблюдаемых несколькими оценщиками одновременно
Выгорание
Синдром эмоционального выгорания (burnout) — состояние хронического стресса на работе: эмоциональное истощение, деперсонализация и снижение профессиональной эффективности
Гештальт
Гештальт-терапия — метод психотерапии, фокусирующийся на осознанности «здесь и сейчас», незавершённых ситуациях и восстановлении контакта с собственными потребностями
Психоанализ
Метод психотерапии и теория психики Фрейда: работа с бессознательным через свободные ассоциации, анализ сновидений и исследование переноса в отношениях с терапевтом
GROW
Коучинговая модель из 4 шагов: Goal (цель) → Reality (реальность) → Options (варианты) → Will (намерение); один из самых популярных фреймворков для коучинговых сессий
Факторный анализ
Статистический метод выявления скрытых переменных (факторов), объясняющих корреляции между наблюдаемыми показателями; основа создания психологических тестов
Батарея Лурии
Нейропсихологическая батарея тестов Александра Лурии — оценивает высшие психические функции (память, речь, гнозис, праксис) при органических поражениях мозга
Когнитивные искажения
Систематические ошибки мышления (катастрофизация, чёрно-белое мышление, чтение мыслей), которые поддерживают тревогу и депрессию; ключевая мишень в КПТ
Привязанность
Теория привязанности Боулби — описывает паттерны эмоциональных связей с близкими (надёжная, тревожная, избегающая); основа понимания отношений в психотерапии

Нейропсихология — изучает связь мозга и поведения, когнитивных функций

Фармация

TDM
Therapeutic Drug Monitoring — контроль концентрации лекарства в крови пациента для подбора оптимальной дозы; особенно важен для антибиотиков, противосудорожных и иммунодепрессантов
Micromedex
Профессиональная база данных лекарственных взаимодействий, дозировок и токсикологии — стандарт для клинических фармацевтов и врачей
Lexicomp
Клиническая база данных лекарств с актуальными дозировками, взаимодействиями и рекомендациями для особых групп пациентов (дети, беременные, почечная недостаточность)
GMP
Good Manufacturing Practice — стандарт производства лекарств: требования к оборудованию, персоналу, контролю качества и документации на фармацевтических предприятиях
CTD
Common Technical Document — унифицированный формат регуляторного досье для регистрации лекарств в FDA, EMA и других агентствах; состоит из 5 модулей
ICH
International Council for Harmonisation — международная организация, разрабатывающая единые стандарты (гайдлайны) для разработки и регистрации лекарств
EMA
European Medicines Agency — регуляторный орган ЕС, отвечающий за оценку и надзор за лекарственными препаратами на территории Евросоюза
FDA
Food and Drug Administration — регуляторный орган США, контролирующий безопасность и эффективность лекарств, продуктов питания и медицинских устройств
ADME
Absorption, Distribution, Metabolism, Excretion — четыре процесса, определяющие судьбу лекарства в организме: всасывание, распределение, метаболизм и выведение
HTS
High-Throughput Screening — автоматизированный скрининг тысяч соединений в поиске активных молекул-кандидатов для разработки новых лекарств
OTC
Over The Counter — безрецептурные препараты, которые можно купить в аптеке без назначения врача; в России маркируются как «без рецепта»
RAC
Regulatory Affairs Certification — международная профессиональная сертификация специалиста по регуляторным вопросам в области лекарств и медицинских изделий
Фармаконадзор
Pharmacovigilance — система мониторинга безопасности лекарств после их выхода на рынок: сбор и анализ сообщений о побочных эффектах
ChEMBL
Открытая база данных биоактивных молекул от Европейского института биоинформатики (EMBL-EBI) — содержит данные о связывании соединений с биологическими мишенями
Антибиотикостерегущая программа
Antibiotic Stewardship Program — больничная программа рационального использования антибиотиков для снижения резистентности и улучшения исходов лечения
Валидация
Documented процесс подтверждения того, что метод, процесс или система стабильно производят результат, соответствующий заданным требованиям; обязательна по GMP
Дженерик
Воспроизведённый лекарственный препарат, содержащий то же активное вещество в той же дозировке, что и оригинальный; выходит после истечения патентной защиты
Фармакодинамика
Наука о том, как лекарство воздействует на организм: механизм действия, рецепторы, дозо-эффектная зависимость — в отличие от фармакокинетики (что организм делает с лекарством)
Биодоступность
Доля введённой дозы лекарства, которая достигает системного кровотока в неизменённом виде; у внутривенных препаратов — 100%, у таблеток — значительно меньше
NDA
New Drug Application — заявка на регистрацию нового лекарства в FDA (США); содержит все данные о безопасности, эффективности и качестве препарата
IND
Investigational New Drug — разрешение FDA на проведение клинических испытаний нового лекарственного вещества на людях
Вспомогательное вещество
Excipient — неактивный компонент лекарственной формы (наполнители, красители, консерванты), обеспечивающий стабильность, растворимость и удобство применения
Молекулярный докинг
Компьютерный метод предсказания связывания молекулы-кандидата с белком-мишенью — позволяет виртуально отбирать потенциальные лекарства до синтеза
Клинические испытания
Исследования на людях, проводимые в 4 фазах для подтверждения безопасности и эффективности нового лекарства перед его регистрацией
Фармакокинетика
Как лекарство движется по организму: всасывание, распределение, метаболизм, выведение (ADME)
DrugBank
База данных лекарственных препаратов и их мишеней: структуры, фармакокинетика, мишени-белки — незаменима в фармакоинформатике
Pharmacovigilance
Фармаконадзор — мониторинг безопасности лекарств на рынке, сбор сообщений о нежелательных реакциях
Antibiotic stewardship
Антибиотикостерегущая программа — рациональное применение антибиотиков для предотвращения резистентности
Validation
Валидация — документальное подтверждение того, что процесс или метод стабильно производит соответствующий продукт
Generic drug
Дженерик — препарат с тем же действующим веществом, что и оригинал, выпускаемый после истечения патента
Pharmacodynamics
Фармакодинамика — изучает механизм действия лекарств на организм
Bioavailability
Биодоступность — доля препарата, достигающая системного кровотока и оказывающая фармакологическое действие
Excipient
Вспомогательное вещество — компонент лекарственной формы кроме активного вещества
Molecular docking
Молекулярный докинг — компьютерное моделирование связывания молекулы лекарства с белком-мишенью
Clinical trials
Клинические испытания — систематическое изучение безопасности и эффективности препарата или метода лечения на людях
Pharmacokinetics
Фармакокинетика — изучает, как организм усваивает, распределяет и выводит лекарство

ИФА
Иммуноферментный анализ (ELISA) — метод обнаружения белков и антител с помощью ферментных меток; основа диагностики инфекций и биомаркеров

LaTeX
Система профессиональной вёрстки научных текстов: стандарт оформления формул, диссертаций и статей в физике, математике, computer science
Zotero
Бесплатный менеджер библиографии: хранит источники, автоматически форматирует ссылки в нужном стиле (APA, ACS, Vancouver и др.)

Препринт
Рукопись статьи, выложенная публично до прохождения рецензирования — позволяет быстро сообщить результаты сообществу (arXiv, bioRxiv)
DOI

Microlearning
Микрообучение — подача знаний короткими сфокусированными блоками по 3–5 минут
ABA therapy
АВА-терапия — прикладной анализ поведения: доказательный метод коррекции поведения при РАС

Art therapy

Fundraising

Datawrapper

Flourish

Storytelling
Сторителлинг — передача информации через нарративные структуры для вовлечения аудитории
Podcast
Подкаст — аудиопрограмма, распространяемая через интернет по подписке
Multimedia
Мультимедиа — контент, объединяющий текст, изображения, аудио и видео

Media monitoring
Медиамониторинг — систематическое отслеживание упоминаний бренда или темы в СМИ

Earned media
Заработанные медиа — бесплатные публикации о компании в СМИ, полученные не за деньги, а за интересную историю или экспертизу
GR

SCCT

Dark site

Sentiment

Кризисные коммуникации — стратегическое управление информацией для защиты репутации в кризис
