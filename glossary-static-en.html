Career Glossary: Professional Terms Explained

753 terms from IT, medicine, science, engineering, psychology, pharmacy and education — explained in plain English. Also available in Russian and 9 languages interactively.

IT

API
Application Programming Interface — how programs communicate with each other over the internet
SPA
Single Page Application — a website that doesn't fully reload, like Gmail or Trello
CSS
Cascading Style Sheets — describe how a page looks: colors, fonts, spacing
HTML
HyperText Markup Language — defines the structure of a webpage: headings, buttons, links
SQL
Structured Query Language — used to query databases. Example: SELECT * FROM users
REST
Representational State Transfer — an architectural style for how APIs communicate
CI/CD
Continuous Integration/Delivery — automatic testing and deployment on every code change
SRE
Site Reliability Engineering — engineers who keep services running 24/7 without downtime
IaC
Infrastructure as Code — servers and networks defined in code files, not set up manually
SLO
Service Level Objective — reliability target, e.g. 99.9% uptime
ORM
Object-Relational Mapping — library letting you work with a DB without writing SQL
Agile
Flexible development methodology using short iterations (sprints)
Scrum
Agile framework: 2-week sprints, daily standups, retrospectives at sprint end
TypeScript
JavaScript with strict types — makes code more reliable and easier to maintain
Redux
State management library for React — stores all app data in one central place
Backlog
Prioritized list of all product tasks not yet completed
DevOps
Culture and practices uniting development and operations for fast, reliable deployment
Docker
Tool for running apps in isolated containers — works the same everywhere
Kubernetes
System for managing many Docker containers at scale
Pixel-perfect
Implementing a design exactly as the designer created it — every pixel in place
Git
Version control system — records code history and enables team collaboration
Linux
Open-source OS — powers most servers and is the foundation of DevOps tooling
Python
Programming language #1 for Data Science, ML and automation — very beginner-friendly syntax
JavaScript
The language of the browser — makes websites interactive; also runs on servers via Node.js
React
JavaScript UI library by Meta — the most popular frontend framework
Vue
JavaScript framework for building UIs — popular alternative to React, widely used in Europe and Asia
Zustand
Tiny, simple state management for React — a Redux alternative without the boilerplate
GitHub
Platform for hosting code and collaborative development built on Git
GraphQL
Alternative to REST for APIs — the client specifies exactly what data it needs
WebSocket
Protocol for real-time two-way connections — used in chats, games, trading platforms
PWA
Progressive Web App — a website that works like a native app: offline, home screen icon
SSR
Server-Side Rendering — page is generated on the server, loads faster and better for SEO
Kanban
A visual workflow management system using cards or signals to control production flow and inventory replenishment
Wireframe
UI skeleton — a schematic layout without colors or fonts, just structure
Codeforces
Competitive programming platform with algorithm and data structure challenges
LeetCode
Coding challenge platform for interview prep — focuses on algorithms and data structures
Jira
Issue tracking tool — teams use it to manage bugs, tasks, and sprint boards
GitHub Actions
Built-in CI/CD in GitHub — automatically runs tests and deploys code on each push
Grafana
Monitoring and dashboard tool — shows real-time server metrics and graphs
Telegram
Messaging app with a powerful Bot API — great for automation and notifications
Discord
Chat platform popular with gamers and developers — supports voice channels and bots

Design

UI
User Interface — the visual part of a product: buttons, forms, icons, menus
UX
User Experience — how easy and pleasant a product is to use
Figma
Vector design tool for UI mockups and prototypes — the industry standard for design teams
Canva
Online design editor for non-designers — templates for posts, presentations, logos
Rive
Interactive animation tool for web and mobile — more powerful than Lottie
Lottie
Lightweight animation format for apps — exported from After Effects, smaller than video
Framer
Interactive prototyping tool with code — sits between Figma and real development
After Effects
Adobe tool for animations, visual effects, and motion design
Miro
An online whiteboard for teamwork — you draw diagrams, sticky notes, and flowcharts together in a browser
FigJam
Figma's built-in whiteboard for workshops, flows, and collaborative planning
Maze
UX testing platform — send prototypes to users and get behavioral analytics
GSAP
GreenSock Animation Platform — powerful JavaScript library for smooth animations

Creative

CMYK
Cyan Magenta Yellow Key — color model for printing. Different from RGB: printers and screens reproduce color differently
RGB
Red Green Blue — color model for screens. All colors are created by mixing three light channels
RAW
Unprocessed camera file — contains maximum data, requires editing in Lightroom or Capture One
AIDA
Attention Interest Desire Action — classic copywriting formula: grab attention, spark interest, build desire, prompt action
CTA
Call To Action — a button, headline, or phrase that tells users what to do next
ISO
Camera sensor sensitivity to light. High ISO = more noise, low ISO = cleaner image
ArtStation
Professional portfolio platform for artists, illustrators, and concept artists
Storyboard
A series of drawings showing scene sequence before shooting or animating
Reels
Short vertical videos on Instagram — the algorithm promotes them organically better than other formats
Turnaround
Character rotation sheet — front, side, and back views for animators and 3D modelers
Vector
Image described with math (Bezier curves) — scales without losing quality. Formats: SVG, AI, EPS
Raster
Pixel-based image — loses quality when enlarged. Formats: JPG, PNG, PSD
Moodboard
A collage of references: photos, colors, textures, fonts. Shows the style direction before work begins
Brand Guidelines
Document with rules for using visual identity: logo, colors, fonts, spacing
Kerning
Spacing between individual letters — important for readability and typographic aesthetics
Dieline
Flat packaging template with cut and fold lines. Used for printing and assembling boxes
SEO copy
Text optimized for search queries — contains keywords and structure to rank in Google
Reach
The number of unique people who saw a post. Different from views/impressions
Engagement Rate
Percentage of your audience that interacts with content: likes, comments, shares
Content plan
Editorial calendar: topics, formats, dates — helps run social media systematically
Hook
The first 3 seconds of a video or opening line of text that grabs attention and keeps people watching
Bokeh
Blurred background in a photo. Achieved with a wide aperture (low f-number)
Exposure
Amount of light hitting the sensor — depends on shutter speed, aperture, and ISO
Retouching
Photo editing: removing blemishes, smoothing skin, color correction in Photoshop
Color Grading
Artistic color processing: creating mood through warm/cool tones

Gamedev

GDD

FPS

Unity
2D/3D game engine used in Hollow Knight, Cuphead, Pokémon GO — widely used by indie studios
Godot
Free open-source game engine — great alternative to Unity for beginners
Unreal Engine
Game engine by Epic Games — powers Fortnite, God of War, Hogwarts Legacy
Blueprint
Visual scripting language inside Unreal Engine — build logic without writing code
Lumen
Unreal Engine 5 global illumination system — realistic real-time shadows and reflections
Nanite
Unreal Engine 5 geometry technology — renders billions of polygons without losing FPS
C#
Programming language by Microsoft — used in Unity, similar to Java
C++
Powerful programming language — used in Unreal Engine, system software, and games
AAA
Triple-A — big-budget games with large teams: GTA, Cyberpunk, FIFA
itch.io
Platform for indie games — publish for free, accept donations, run game jams
GameMaker
2D game engine — Undertale, Hotline Miami, and Spelunky were made with it
Asset Store
Unity's marketplace — buy or download models, animations, sounds, and scripts
GDC
Game Developers Conference — the largest dev conference; many talks are free online
Roblox Studio
Built-in editor for creating games inside Roblox — monetizable with a huge audience
SOTA
State Of The Art — the best current result on a given benchmark or task

Security

CTF
Capture The Flag — cybersecurity competitions where you legally solve hacking challenges
SIEM
Security Information and Event Management — security threat monitoring system
OWASP
Open Web Application Security Project — publishes top web app vulnerabilities
Pentest
Penetration Test — legally hacking a system to find vulnerabilities before real hackers
bug bounty
Reward program for finding vulnerabilities — companies pay ethical hackers for findings

A gene editing technology that allows precise changes to DNA sequences — applied in biotechnology and medicine
GCP
Good Clinical Practice — the international standard for conducting clinical trials, protecting participants' rights
PubMed
The largest free database of biomedical and life science research articles from the US National Library of Medicine
BLAST
Basic Local Alignment Search Tool — searches databases for similar DNA and protein sequences
UpToDate
A clinical decision-support tool with practice guidelines — doctors use it at the bedside
Benchling
A cloud platform for electronic lab notebooks and biotech experiment management

EHR
Electronic Health Record — a digital version of a patient's medical history stored in a clinical system
ICD-10
International Classification of Diseases, 10th revision — the universal language of diagnoses used by doctors worldwide
Anamnesis
Patient medical history: complaints, symptoms, past conditions — gathered by a doctor at the initial visit
Differential diagnosis
A systematic process of ruling out diseases with similar symptoms to arrive at the single correct diagnosis
CBT
Cognitive Behavioral Therapy — an evidence-based psychotherapy that targets unhelpful thoughts and behaviours; the gold standard for anxiety and depression
RCT
Randomized Controlled Trial — the gold standard of clinical research: patients are randomly split into groups
PCR
Polymerase Chain Reaction — a method for analysing DNA/RNA; used to diagnose infections and genetic conditions
DNA sequencing
Determining the order of nucleotides in a DNA or RNA molecule — the foundation of personalised medicine
Bioinformatics
A science combining biology and programming: genomic data analysis, searching BLAST/NCBI databases, protein structures
Biomarker
A measurable biological indicator (protein level, genetic mutation) that signals a disease or treatment response
Placebo
A dummy treatment with no active ingredient — used in clinical trials as a control
Pharmacokinetics
The study of what the body does to a drug: how it is absorbed, distributed, metabolised, and excreted over time
Evidence-based medicine
Making clinical decisions based on the best available scientific evidence, not just clinical experience alone
Telemedicine
Delivering medical care remotely — video consultations, remote patient monitoring via wearables
Robot-assisted surgery
Operations using the da Vinci robot — the surgeon controls manipulators with high precision; fewer complications, faster recovery
Motivational interviewing
A counselling technique that helps patients arrive at their own decision to change behaviour — quit smoking, lose weight, etc.
Multidisciplinary team
MDT — a group of specialists from different fields (physician, surgeon, oncologist, psychologist, nurse) jointly managing a complex patient
Residency
Clinical residency — 2–6 years of supervised specialised practice after obtaining a medical degree
MedTech
Medical Technology — the industry applying technology to medicine: diagnostic devices, wearable sensors, AI for imaging analysis, e-prescriptions
Q1 journal
A scientific journal in the first impact-factor quartile — publishing there signals high peer-review quality and wide readership

Cochrane Library — the largest database of systematic reviews and meta-analyses: synthesises the best evidence for clinical practice
Biostatistics
The application of statistical methods to biomedical data: group comparisons, p-values, confidence intervals, survival analysis

EU Medical Device Regulation — European regulation for medical devices; CE marking grants the right to sell a device in the EU
CONSORT
Consolidated Standards of Reporting Trials — the international standard for reporting randomised clinical trials
Impact factor
A measure of a journal's citation rate — the higher it is, the more influential the journal and the harder it is to publish there
Systematic review
A scientific paper that combines and critically appraises all research on a specific question — the highest level of evidence
Meta-analysis
A statistical synthesis of results from multiple studies — yields more precise conclusions than any single study
h-index
A bibliometric indicator: a researcher with h=20 has published ≥20 papers each cited ≥20 times

Ecology

ArcGIS
Professional GIS platform by Esri — the industry standard for geospatial analysis and cartography
QGIS
Free open-source GIS — a full-featured alternative to ArcGIS widely used by ecologists and researchers
Google Earth Engine
Google's cloud platform for analysing petabytes of satellite imagery — monitors forests, ice sheets, and water bodies
Sentinel-2
European Earth observation satellite with 10 m resolution — used for vegetation and deforestation monitoring
ERA5
Climate reanalysis dataset from Copernicus/ECMWF — hourly global weather data from 1940 to present
CMIP6
The international set of climate model outputs used in IPCC assessment reports
IPCC

rasterio
A Python library for reading and processing raster geospatial data (GeoTIFF, NetCDF)
xarray

GIS

Remote sensing
Collecting information about Earth's surface from satellites or aircraft without direct contact

The global list of species threat levels: from LC (Least Concern) to EX (Extinct)
Carbon footprint
The total CO₂ emissions generated by a person, company, or product

Biocenosis
The community of all living organisms in a specific habitat: plants, animals, fungi, and microbes together
Biodiversity
The variety of species, ecosystems, and genes on Earth — the primary measure of ecosystem health

A genome editing system: a guide RNA leads the Cas9 protein to a specific DNA site where it makes a cut — enabling gene knockout, correction, or insertion
NGS

Pharmacy

TDM
Therapeutic Drug Monitoring — measuring drug concentration in a patient's blood to optimise dosing; especially important for antibiotics, anticonvulsants, and immunosuppressants
Micromedex
A professional database of drug interactions, dosing, and toxicology — the standard reference for clinical pharmacists and physicians
Lexicomp
A clinical drug database with current dosages, interactions, and recommendations for special populations (children, pregnant women, renal impairment)
GMP
Good Manufacturing Practice — the standard for drug production: requirements for equipment, personnel, quality control, and documentation at pharmaceutical facilities
CTD
Common Technical Document — the standardised format for a regulatory dossier when registering drugs with the FDA, EMA, and other agencies; consists of 5 modules
ICH
International Council for Harmonisation — the international body that develops unified guidelines for pharmaceutical development and drug registration
EMA
European Medicines Agency — the EU regulatory authority responsible for evaluating and supervising medicines across the European Union
FDA
Food and Drug Administration — the US regulatory authority overseeing the safety and efficacy of drugs, food, and medical devices
ADME
Absorption, Distribution, Metabolism, Excretion — the four processes that determine a drug's fate in the body: how it is absorbed, distributed, broken down, and eliminated
HTS
High-Throughput Screening — automated screening of thousands of compounds to identify active candidate molecules for new drug development
OTC
Over The Counter — non-prescription medicines that can be purchased at a pharmacy without a doctor's prescription
RAC
Regulatory Affairs Certification — an international professional certification for regulatory affairs specialists in drugs and medical devices
ChEMBL
An open bioactive molecule database from the EMBL European Bioinformatics Institute — contains binding data for compounds against biological targets
NDA
New Drug Application — a submission to the FDA requesting approval to market a new drug in the USA; contains all safety, efficacy, and quality data
IND
Investigational New Drug — FDA authorisation to begin clinical trials of a new drug substance in humans
DrugBank
A database of drugs and their targets: structures, pharmacokinetics, protein targets — indispensable in pharmacoinformatics and drug discovery
Pharmacovigilance
The science and activities related to detecting, assessing, and preventing adverse effects of medicines after they reach the market
Antibiotic stewardship
A hospital programme for the rational use of antibiotics to reduce resistance and improve patient outcomes
Validation
A documented process confirming that a method, process, or system consistently produces results meeting specified requirements; mandatory under GMP
Generic drug
A medicine containing the same active substance at the same dose as the originator drug; comes to market after the original patent expires
Pharmacodynamics
The study of how a drug affects the body: mechanism of action, receptors, and dose-response relationships — as opposed to pharmacokinetics (what the body does to the drug)
Bioavailability
The fraction of an administered dose that reaches the systemic circulation unchanged; 100% for intravenous drugs, considerably less for oral tablets
Excipient
An inactive ingredient in a drug formulation (fillers, colourants, preservatives) that ensures stability, solubility, and ease of use
Molecular docking
A computational method for predicting how a candidate molecule binds to a protein target — enables virtual screening of potential drugs before synthesis
Clinical trials
Studies conducted in humans across 4 phases to confirm the safety and efficacy of a new drug before regulatory approval
Pharmacokinetics
The study of what the body does to a drug: how it is absorbed, distributed, metabolised, and excreted over time
